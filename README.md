# DRHP RAG — SK Minerals & Additives Limited

A retrieval-augmented generation (RAG) system for querying the Draft Red Herring Prospectus (DRHP) of SK Minerals & Additives Limited via a FastAPI API.

---

## How it works

```
PDF → ingest.py (extract pages) → chunk_by_sections()
    → embed.py (all-MiniLM-L6-v2) → ChromaDB
    → generate.py (top 5 chunks → Claude via OpenRouter)
    → grounded answer with citations
```

---

## Project structure

```
s45-rag/
├── ingest.py      — PDF text extraction and section chunking
├── embed.py       — Embedding + ChromaDB vectorstore build/query
├── generate.py    — Answer generation via Claude (OpenRouter)
├── app.py         — FastAPI app (POST /upload, POST /query)
├── view.py        — CLI viewer for extracted pages
├── chroma_db/     — Persisted ChromaDB vectorstore (auto-created)
└── README.md
```

## Setup

```bash
pip install pdfplumber sentence-transformers chromadb openai fastapi uvicorn python-multipart

# Build the vectorstore from the DRHP PDF
python embed.py

# Start the API
uvicorn app:app --reload
# → http://localhost:8000
```

---

## API

### `POST /upload`
Upload a PDF and build its vectorstore.

```bash
curl -X POST http://localhost:8000/upload \
  -F "file=@DRHP_20250430105517.pdf"
```

```json
{
  "message": "PDF processed and vectorstore built successfully.",
  "pages": 470,
  "chunks": 719
}
```

### `POST /query`
Ask a question. Returns a grounded answer with section + page citations.

```bash
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{"question": "Who are the promoters?"}'
```

```json
{
  "question": "Who are the promoters?",
  "answer": "The promoters are Mrs. Sunita Rani, Mr. Mohit Jindal, Mr. Rohit Jindal and Mr. Shubham Jindal. [Section: OUR PROMOTERS, Page: 25]",
  "chunks_used": 5
}
```

### `GET /`
Health check.

```json
{"status": "ok", "endpoints": ["POST /upload", "POST /query"]}
```
